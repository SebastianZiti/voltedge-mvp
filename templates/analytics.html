ForecastService

The MVP uses a simple Python domain service. It looks at recent occupied charger telemetry, calculates average load and applies a small growth factor. This keeps the model explainable for the exam while still demonstrating predictive analytics.

In the report this maps to the Analytics bounded context and the LoadForecastCalculated domain event. In the code it is intentionally implemented as plain Python functions so the team can explain it line by line.
